## .env file setup

 ### Connect your local database (mongoDB atlas)

- USER_NAME = xxx
- USER_PASS = xxx


### Stripe Secret key
- STRIPE_SECRET_KEY = xxx

### Access token
- ACCESS_TOKEN_SECRET= xxx

### CMS payment system
- STORE_ID= xxx
- STORE_PASS= xxx


## Project setup

### Follow the process

- npm i (for install all packages)
- code . (for open code)
- nodemon index.js or node index.js  (for run server in local host)
